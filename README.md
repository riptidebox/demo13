# ООО "Ткани"

ООО «Ткани»  - магазин по продаже тканей и фурнитуры в Кропоткине. 

## Способ запуска

Для запуска приложения необходимо запустить файл demo5.exe

## Средства разработки 

* СУБД - MySql Workbench
* Язык программирования - C#
* Среда разработки - Visual Studio 2022
